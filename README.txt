La configuración de la base de datos es:
    host     : 'localhost',
    user     : 'root',
    database : 'ada',
    password : 'admin'

En documentation se encuentra el dump de la tabla de la base de datos de personas, donde existen 3 tipos, 0 para administrador, 1 para profesor, y 2 para alumno.



Para poder correr el sistema ADA, es necesario primero realizar una instalación de los modulos(se realiza durante la primera vez), para esto, es necesario correr el comando npm install, posteriormente se utiliza npm start para correr el sistema.
